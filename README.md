১. getElementById, getElementsByClassName এবং querySelector / querySelectorAll-এর মধ্যে পার্থক্য কী?

উত্তর : ০১ —
getElementById: এটি HTML collection এর মধ্যে ইউনিক বা নির্দিষ্ট কাউকে খোঁজার জন্য ব্যবহৃত হয়।

getElementsByClassName: এটি সাধারণত একই নামে অনেকগুলো ক্লাস থাকলে সবকটি এলিমেন্টকে একটি লিস্ট আকারে খুঁজে দেয়।

querySelector: কুয়েরী সিলেক্টর সাধারণত প্রথম এলিমেন্টকে খুঁজে দেয়। কিন্তু এটি ব্যবহার করার আগে CSS সিলেক্টর ব্যবহার করতে হবে। যেমন যদি সেটি আইডি হয় তাহলে প্রথমে (#) চিহ্ন দিয়ে খুঁজতে হবে আর যদি ক্লাস হয় তাহলে শুরুতে (.) ডট চিহ্ন দিয়ে খুঁজতে হবে।

querySelectorAll: সাধারণত এটি ব্যবহার করলে সবগুলো এলিমেন্টকে একটি লিস্ট হিসেবে দিয়ে দেয়।


২. DOM-এ কীভাবে একটি নতুন এলিমেন্ট তৈরি এবং ইনসার্ট (যুক্ত) করতে হয়?
উত্তর : ০২ —
DOM এ নতুন কোনো এলিমেন্ট তৈরি ও যুক্ত করতে সাধারণত document.createElement('যে tag-এর মধ্যে রাখব তার নাম দিতে হবে') তারপর appendChild() বা append() ব্যবহার করে ওই tag-এর মধ্যে ঢুকিয়ে দিতে হবে।

৩. Event Bubbling কী? এবং এটি কীভাবে কাজ করে?
উত্তর : ০৩ —
ইভেন্ট বাবলিং হলো যখন কোনো একটি এলিমেন্ট বা বাটনে ক্লিক করা হয়, তখন ক্লিক event টি বাটনের মধ্যে সীমাবদ্ধ না থেকে আস্তে আস্তে উপরের দিকে উঠে আসে।

৪. জাভাস্ক্রিপ্টে Event Delegation কী? এটি কেন দরকারী?
উত্তর : ০৪ —
ইভেন্ট ডেলিগেশন হলো প্রতিটি চাইল্ড এলিমেন্টকে আলাদা আলাদা ইভেন্ট লিসেনার যোগ না করে, তাদের প্যারেন্ট এলিমেন্টকে একটি মাত্র ইভেন্ট লিসেনার সেট করা।
